# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [11.0.0] - 2026-02-01

### Added
- 20+ indicateurs techniques avancés (ADX, Stochastic RSI, Williams %R, CMF, OBV, VWAP, SuperTrend, Squeeze)
- Scoring multi-horizon (Intraday, Swing, Position)
- Machine Learning ensemble (RandomForest, GradientBoosting, IsolationForest, AdaBoost)
- Scanner Breakout avec détection de cassures
- Volume Spike Alert (> 200% moyenne)
- Pattern Reversal detection (Hammer, Engulfing, Doji)
- Support marchés européens et crypto

### Changed
- Amélioration performance du scanner
- Optimisation des requêtes API

### Fixed
- Correction calcul RSI sur petites séries
- Fix timezone pour marchés européens

## [10.0.0] - 2026-01-15

### Added
- Support initial multi-marchés
- Indicateurs techniques de base
