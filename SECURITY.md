# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 11.x    | :white_check_mark: |
| < 11.0  | :x:                |

## Reporting a Vulnerability

Si vous découvrez une vulnérabilité de sécurité, veuillez :

1. **Ne pas** ouvrir une issue publique
2. Envoyer un email à [security@example.com] avec :
   - Description de la vulnérabilité
   - Étapes pour reproduire
   - Impact potentiel

Nous nous engageons à :
- Accuser réception sous 48h
- Fournir une estimation de correction sous 7 jours
- Vous tenir informé de la résolution

## Responsible Disclosure

Nous pratiquons la divulgation responsable. Merci de nous laisser le temps de corriger avant toute publication.
